OnStep Aux INDI Driver V0.1
-------------------------------------------------

An INDI driver for OnStep when used without a telescope(mount) (https://onstep.groups.io/g/main/wiki)
  Copyright (C) 2025 Ed Lee

Supports the following OnStep functions/features:

Focuser
Rotator
Weather sensors
Auxiliary features:
    Switches
    Dew Heaters
    Intervalometers
    USB Ports via OnStepX plugin (https://github.com/hjd1964/OnStepX-Plugins/tree/main/usb)

Version 0.1
    Initial test release
Version 0.2
    Fix for Focuser absolute move
